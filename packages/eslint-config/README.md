## Getting Started

Install `eslint-config-jolt` using `yarn`:

```shell
yarn add --dev eslint-config-jolt
```

## Usage

In your `.eslintrc`, add the following:

```json
"extends": "jolt"
```

If you prefer to have your config in your `package.json`, add:

```json
"eslintConfig": {
  "extends": "jolt"
},
```
